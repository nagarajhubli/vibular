About This Site

A love letter to Angular, written mostly in prompts. The code is real; the effort is suspiciously low.

Why "Vibular"?

Because Angular is a serious framework, and this is a decidedly unserious rebrand of it. Vibe + Angular → Vibular. It sounds almost anatomical, slightly made up, and entirely on-brand for a site assembled by asking nicely instead of typing carefully. If you squint, it's a framework. If you don't, it's still fine.

How It Was Created

terminal Scaffolded with Angular CLI npx @angular/cli@latest new — generated the project structure, routing, and SCSS support. palette Angular Material added via schematic ng add @angular/material — injected the Material 3 theme into styles.scss and index.html. pages Pages generated via CLI ng generate component pages/home and ng generate component pages/about — created the routed page components.

Frameworks & Libraries

Angular 21 Frontend Framework The framework that introduced us to dependency injection and never quite let us leave. Standalone components, signals, new control flow — same old love. Angular Material 21 UI Component Library Google's design system, ported to Angular, now in its Material 3 era. It has feelings (CSS variables) and boundaries (density tokens). Respect both. TypeScript 5.9 Language JavaScript with adult supervision. Yells at you at compile time so the browser doesn't have to at runtime. You'll thank it later. SCSS Stylesheet Language CSS, but it finally learned how to nest. Angular Material's theming lives here, which is why mat.theme() feels like magic and mostly is. Vitest 4 Testing Framework Tests so fast you'll almost want to write them. Karma is in a nice retirement home now; Vitest does the cardio.
